# Data-Intelligence-Bootcamp-Assignment-1
A Student Performance Management System
- The system will store student information such as Roll Number, Name, Branch, and 
  Subject-wise Marks. 
- It will support basic operations like adding, viewing, updating, and deleting records. 
- The system will automatically assign grades based on marks. 
- It will provide analytical features such as class average and performance-based filtering. 
- The project will be developed as a console-based application in Python.

Key Features
- Add new student records 
- View all student records 
- Search student by Roll Number or Name 
- Update existing student information 
- Delete student records 
- Calculate class average 
- Automatically assign grades 
- Filter students by grade category (Distinction, First Class, Second Class, Fail) 
- Find class topper(s)
